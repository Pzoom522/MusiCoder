市场前景

  • 消费级娱乐市场
普通用户也可以自主完成复杂的混音操作，像使用照片滤镜一样，自由地变换音乐质感；

（参照：Prisma-ai                                  ）

  • 专业级市场
创作者可以敏捷地实现创意，获得其它软件不能生成的音效，从而扩展灵感；
  • 学术研究领域
研究者可以利用我们的API，进行音乐质感合成与可视化。
